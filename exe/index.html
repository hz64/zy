# EXE

EXE File英文全名executable file ，译作可执行文件，可移植可执行 (PE) 文件格式的文件，它可以加载到内存中，并由操作系统加载程序执行，是可在操作系统存储空间中浮动定位的可执行程序。如记事本程序notepad.exe ，可以用来编辑文档，如：测试.txt双击打开notepad.exe记事本程序来进行编辑处理
